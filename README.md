ERP 沙盘虚拟运营平台
====================

An applicaiton built with Marionette.js.

## 开始
[Clone]() or [Download]() this repo.

```sh
git clone && cd 
```

确认 [Node.js](http://nodejs.org/) & [npm](https://www.npmjs.org/) 已经
[installed](http://nodejs.org/download/).
建议使用 [nvm]().

```sh
npm install 
npm install gulp -g
gulp watch
```

等待 2/3 秒完成编译, 浏览器会自动打开 http://localhost:3000.

## 指引
